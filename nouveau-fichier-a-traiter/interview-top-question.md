# Interview top questions - sources web

Recherche web verifiee le 2026-03-27.

Objectif:
- centraliser des listes de questions d'entretien pour construire une future bulle de savoir
- couvrir les technos deja ciblees dans ce fichier
- separer les ressources par niveau: debutant, avance, pro, expert

Regle de classement:
- si la source annonce directement Beginner / Intermediate / Advanced / Expert, je reprends ce decoupage
- sinon, le niveau ci-dessous est une inference basee sur les sujets couverts
- priorite pour l'app: les sources qui ont deja une separation claire des niveaux ou des sections "freshers / experienced / scenario-based"
